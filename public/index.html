Projects

Qualbum

This project tries to solve a specific problem I've been having. Every time I try to merge the pictures from my phone/shared folder I have to go to the process of:

  • Remove duplicates.
  • Remove what I don't want to keep.
  • Choose which picture I want to keep among similar ones. (Remember when people say "I'll take a couple more just in case? This situation).
  • Create a "summary" collection with less media (Currently I'm not doing this part, but I want this program to help me with that)

Therefore, I want a program to do that for me as automated as possible. And since I wanted to learn how to program GUIs and C#, it was the perfect oportunity.

I realize it would probably be more suited for android, but I wanted to learn C#, so... future work.

Background-switcher

I wanted to have a dynamic desktop image, but not just keep pulling the images from a folder, because you have to remember to keep updating the folder from time to time, and I wanted something with new pictures automatically.

So, I created this small program that runs when my laptop starts that pulls the Picture of the day from a webpage, and places it as my background image for that day. And the best part is that right now is a stable and finished project that Just Works!
